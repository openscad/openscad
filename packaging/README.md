# Packaging

This directory contains packaging specifications for creating distribution-specific packages.

## Planned Packages

### Debian/Ubuntu (packaging/debian/)
- `.deb` package generation
- Control files, dependencies, post-install scripts

### Fedora/RHEL (packaging/fedora/)
- `.rpm` package via spec file
- Dependencies and build requirements
